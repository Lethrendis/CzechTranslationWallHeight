
# CzechWallHeight

Czech translation of Wall Height version 7.0.8

## Popis
Tento modul poskytuje českou lokalizaci pro Wall Height 7.0.8. Obsahuje překlady základních textů aplikace.

## Instalace
Pro instalaci modulu obvyklým upůsobem můžeš použít následující odkaz ve Foundry VTT:

```
https://raw.githubusercontent.com/Lethrendis/CzechTranslationWallHeight/main/module.json
```
